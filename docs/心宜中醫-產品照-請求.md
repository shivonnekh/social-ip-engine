   ```
   static/sales/ointments/
     ointment_chashu_lvdou.png

     ointment_zhihen_shizhen.png
   ```

2. **Update `data/product_catalog.json`** — swap each entry's `image_url` from the healthy-food.hk URL (or `PLACEHOLDER_TO_BE_REPLACED` for ointments) to `/static/sales/<type>/<product_id>.<ext>`. The router serves `/static/*` already via FastAPI StaticFiles mount in `src/web.py:201`.

3. **Remove the 示意圖 disclaimer** from `data/product_catalog.json._meta`:
   ```json
   "image_disclaimer_required_for_types": []  // was ["soup"]
   ```

4. **Update the two card files** (`tcm_paid_soups.json` + `tcm_paid_ointments.json`) `products[].image_url` to match the catalog.

5. **Re-run tests:**
   ```bash
   pytest tests/test_product_recommendation.py
   ```

6. **Commit on a new branch** off `feature/jessica-product-sales-phase3` (or a fresh branch once phase 3 is merged) with message:
   ```
   feat(jessica): replace 示意圖 with official Care Plus product photos
   ```

Time estimate: 30 minutes once photos are received.
