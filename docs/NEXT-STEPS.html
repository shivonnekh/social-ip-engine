步骤做什么在哪里
1给 Jackie 的 Facebook Page 生成 long-lived Page token，权限勾：pages_messaging、pages_manage_engagement、pages_read_engagementMeta App dashboard
2Messenger webhook 回调填 https://tcm-jessica.onrender.com/webhook/facebook（verify token 用现有的 META_VERIFY_TOKEN），Page 订阅 messages + feedMeta App dashboard
3填 4 个环境变量：FB_PAGE_ID、FB_PAGE_ACCESS_TOKEN、FB_PAGE_LANGUAGE=en（Jackie 是英文号，这个必填，不填带语言标签的规则不会发）、（可选）FB_APP_SECRETRender dashboard → social-ip-engine → Environment
4最后才把 FB_ENABLED=true（dashboard 里设，永远不要写死进 render.yaml）Render dashboard
5验证：终端跑 python3 scripts/fb_probe.py（期望输出 ignored；出 disabled = 第 4 步没生效）→ 发一条测试关键词评论 → 发一条真问题私信本地终端 + FB
回滚开关：出任何问题，Render dashboard 把 FB_ENABLED=false 即刻止血，IG 完全不受影响。
